# ------------------------------------------------PROCESSING BUNNY GAME PROJECT------------------------------------------------------------------ 

This project uses Processing language to create a Bunny Game . The bunny can jump up and down to avoid colliding with obstacles.

Table of content

1. Confiuration
2. Installation
3. Operation
4. Manifest
5. Copyright and License
6. Contact Information

1. Configuration
	This project requires Processing application in order to run.  
2. Installation
	a) Open the Processing application.
	b) Run the project by running the "Bunny_Game.pde".
3. Operation	
	Using the navigation buttons on the keyboard to control the bunny to jump.
4. Manifest (list of files and short description of their roles)
	Bunny_Game.pde
		The whole game is included within this file.
